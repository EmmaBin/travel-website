
# My travelogue 
### An app  where users can manage their trip by searching for destinations, creating checklists, and tracking budgets.

## Technologies used
- Python
- JavaScript
- AJAX/JSON
- Flask
- Jinja
- React JS
- SQL / SQL Alchemy
- Chart.js
- Google Maps API / Google Places API
- HTML
- CSS
